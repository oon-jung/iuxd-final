### 📄 NEW Slide: 자유응답 키워드 분석

**제목:** 숫자가 말하지 않은 것 — Free-Text Keyword Analysis

**본문 (워드클라우드 or 막대 차트):**
```
섹션 5 자유응답 분석 (N=30 중 유효 응답 24건)
빈도 TOP 8 키워드:
1. "여운"/"잔상"      (13건, 54%)   ← Persona 1 직결
2. "허무"/"무력감"    (9건, 38%)
3. "짠하다"/"가엾다"  (7건, 29%)
4. "정적"/"침묵"      (6건, 25%)    ← 사운드 설계 근거
5. "긴장"/"불안"      (5건, 21%)
6. "체험하고 싶다"    (5건, 21%)
7. "나도 지켜주고 싶" (4건, 17%)    ← 단종 은유 수용성 근거
8. "역사가 가깝"      (3건, 13%)
```

**인사이트 박스 (강조):**
> "여운·정적·침묵" 키워드가 상위권을 차지한 것은 **수동적 정보 전달이 아닌, 능동적 감정 체류 공간**을 원한다는 신호다. 이는 본 전시의 **느린 페이싱 + 사운드 부재 구간** 설계 근거가 된다.

**주의:** 위 키워드/빈도는 예시 템플릿. 실제 팀의 자유응답 데이터를 워드클라우드로 시각화 필요.

## 섹션 06. UX Contents Flow — 강화

### 📄 REVISE Slide: 기존 p.10 UX Contents Flow → 논리 링크 추가

**기존 구조 유지하되, 각 섹션 블록 옆에 "→" 화살표로 논리 링크 추가:**

```
#기 [Introduction]
│   └─ WI1 (SNS 좋아요) ──→ "왜 바로 영상으로?" 
│                             동기(좋아요) → 행동(귀가) 전환
│   └─ Video 1 (친구 대화/귀가)
│
#승 [Main Interaction]    "왜 갑자기 피딩?"
│                          → 행동 의도 → 실제 만남 직전 S2 삽입
│   └─ S2 First Meeting (신규, 인내) ──→ 신뢰 형성
│   └─ WI2 (피딩게임, Day1→30)      ──→ 유대 형성
│
#전 [Crisis]               "왜 갑자기 위기?"
│                          → 사적 관계 → 사회적 갈등 진입점 S3 삽입
│   └─ S3 Village Notice (신규, 선택) ──→ 외부 압력 체감
│   └─ Video 2 (독약 사건)            ──→ 관찰자 → 당사자
│
#결 [Resolution]           "왜 단종 이야기가?"
│                          → 현재 사건 → 역사 은유 연결 S4 삽입
│   └─ S4 Two Faces One Grief (신규) ──→ 은유 명시화
│   └─ Video 3 (매장/애도)            ──→ 카타르시스
│   └─ WI3 (텍스트 입력)              ──→ 개인 성찰
│
[Epilogue]                 
│   └─ S5 Memory Wall (신규) ──→ 공동체 기억
```

## 발표 스크립트 권장 포인트 (Presentation Hooks)

발표 때 평가자 피드백에 **선제적으로** 대응할 문장:

### 개회 (30초)
> "지난 중간 평가에서 저희는 **'깊이 부족'**이라는 피드백을 받았습니다. 오늘 발표는 **각 지적 사항에 대한 구체적 응답**으로 구성됩니다."

### 설문 섹션 전환 (30초)
> "단순 비율 나열을 넘어서, **왜 그런 응답이 나왔는지**, 그리고 그것이 **어떤 디자인 결정으로 이어졌는지** 3단 표로 풀었습니다."

### UX Flow 섹션 전환 (20초)
> "이전 플로우는 시간 순서였습니다. 이번 플로우는 **논리 순서**입니다. 각 전환에 '왜 이 다음인가'를 명시했습니다."

### 인터랙션 섹션 도입 (40초)
> "클릭, 드래그, 스크롤은 모든 웹이 합니다. 저희 전시만의 시그니처는 **기다림, 집단의 선택, 이중 노출, 정적, 잔상**입니다. 각각이 기획 의도와 직결됩니다."

### 마무리 (20초)
> "컨셉에서 멈추지 않았습니다. **QR 코드**로 지금 바로 체험하실 수 있습니다."

## 체크리스트 — 개정 완료 판정

개정 후 기획서가 다음 질문에 모두 "예" 답변 가능한지:

- [ ] 설문 결과에 **"왜"**에 해당하는 해설이 있는가?
- [ ] 설문이 **디자인 결정**과 인과 링크로 연결되는가?
- [ ] 자유응답 키워드가 **가시화**되어 있는가?
- [ ] 각 페르소나가 **어느 섹션에서 만족**하는지 표시되는가?
- [ ] 경쟁/유사 사례와의 **차별점**이 명시되는가?
