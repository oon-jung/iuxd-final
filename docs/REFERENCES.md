# 페이지별 레퍼런스 · 에셋 기입 문서

> 이 문서에 **22페이지 전체의 참고 링크 / 이미지 / GIF / 메모**를 한 번에 기입하세요.
>
> ### 사용 방법
> 1. **이미지/GIF 파일**은 해당 페이지 폴더(`public/assets/slides/page-XX-*/`)에 넣고, 파일명을 각 섹션 `파일명` 항목에 적습니다.
> 2. **참고 링크**(다른 사이트/영상/Figma/Pinterest 등)는 `참고 링크` 항목에 URL로 붙여넣습니다. 해당 링크를 어느 슬라이드에 반영할지 따로 명시하지 않아도, 이 섹션 위치가 이미 슬라이드 지정입니다.
> 3. **메모**에는 분위기/톤/원하는 표현 등 자유 기재.
>
> 기입 후 저에게 알려주시면 → 기획서 재작성 + 실제 에셋 반영합니다.
