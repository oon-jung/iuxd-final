# Slide Images

슬라이드별로 이미지를 여기에 드롭하면 코드가 자동으로 집어갑니다.

## 구조

- `page4/01.jpg ~ 09.jpg` — 페이지 4 관계 카드 (9장). 9번이 고양이(main).
- `page5/01.png ~ 06.png` — 페이지 5 장면 카드 (6장). 6번이 main.

## 규칙

- 파일명은 소문자 `01.jpg`, `02.jpg` 식으로 0 패딩 2자리.
- 확장자는 `.jpg`, `.png`, `.webp` 아무거나. 없으면 코드가 그라디언트 폴백으로 표시됩니다.
- 권장 해상도: 2:3 세로 비율, 800 x 1200 이상.
